Problem 2:

Problem 4:

• If the action is "addEnd," add the value to the end of the array using push.
• If the action is "addStart," add the value to the beginning of the array using unshift.
• If the action is "addAtPosition," add the value at the specified position using splice.
• If the action is not recognized, return the original array.

• If the boolean parameter is true, add the elements of the second array to the end of the first array using push.
• If the boolean parameter is false, add the elements of the second array to the beginning of the first array using unshift.
